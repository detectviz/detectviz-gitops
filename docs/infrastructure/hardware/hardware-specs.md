# DetectViz 硬體規格說明

## 系統總覽

DetectViz 平臺採用 ASMB10-iKVM 工業級伺服器，提供高可用性的 Kubernetes 集群運行環境。

## 主要硬體規格

### 處理器 (CPU)
- **型號**：Intel(R) Core(TM) i7-14700F
- **核心數量**：20 核心
- **邏輯處理器**：28 個 (支援超線程)
- **基礎頻率**：2.1 GHz
- **睿頻**：5.4 GHz
- **快取**：33MB Intel Smart Cache

### 記憶體 (RAM)
- **容量**：64 GB (32GB × 2)
- **型號**：D5-6000
- **類型**：DDR5
- **頻率**：6000 MT/s
- **時序**：CL30
- **ECC 支持**：是 (工業級)

### 儲存系統

#### 主系統碟 (SATA SSD)
- **型號**：Acer SSD RE100
- **容量**：512 GB
- **介面**：SATA 6Gb/s
- **讀取速度**：562 MB/s
- **寫入速度**：529 MB/s
- **用途**：Proxmox 系統安裝

#### 高性能資料碟 (NVMe SSD)
- **型號**：TEAM MP44
- **容量**：2 TB
- **介面**：PCIe Gen4
- **讀取速度**：7400 MB/s
- **寫入速度**：7000 MB/s
- **用途**：VM 儲存、資料庫、AI 模型

### 網路介面
- **晶片組**：Intel I210-AT
- **介面數量**：3 個 Gigabit Ethernet
- **主要用途**：
  - `enp4s0`：Proxmox 主管理網路 (vmbr0)
  - `enp5s0`：預留擴展網路
  - `enx36fe0bfce7d0`：預留擴展網路

## IPMI/KVM 管理

### 管理介面規格
- **型號**：ASMB10-iKVM
- **網路**：專用管理網路 (192.168.0.104)
- **支援協議**：
  - IPMI 2.0
  - KVM-over-IP
  - Virtual Media
  - Serial-over-LAN

### 管理功能
- **遠端電源控制**：開機/關機/重啟
- **遠端控制台**：KVM-over-IP
- **感測器監控**：溫度、電壓、風扇
- **日誌記錄**：系統事件和錯誤
- **韌體更新**：線上更新功能

## 電源和冷卻

### 電源供應
- **類型**：工業級電源供應器
- **冗餘**：單電源設計 (建議 UPS 保護)
- **效率**：80 Plus 金牌等級

### 散熱系統
- **風扇**：多個系統風扇
- **散熱片**：CPU 和晶片組專用散熱
- **空氣流**：正面進氣、後方排氣設計

## 機箱和擴展

### 機箱規格
- **類型**：機架式伺服器
- **尺寸**：標準 4U 或更小
- **擴展槽**：PCIe 插槽支援
- **硬碟托架**：2.5"/3.5" 混合支援

### 擴展能力
- **PCIe 插槽**：用於網路卡、GPU 等擴展
- **USB 介面**：前後板各 2 個
- **內部連接**：SATA 和 PCIe 擴展

## 環境要求

### 運行環境
- **溫度**：10°C ~ 35°C (最佳 20°C ~ 25°C)
- **濕度**：20% ~ 80% RH (非凝結)
- **海拔**：最高 3000 公尺
- **振動**：運輸時需保護

### 電源要求
- **電壓**：100-240V AC
- **頻率**：50/60 Hz
- **功率**：根據配置調整
- **保護**：建議 UPS 不斷電系統

## 維修和支援

### 保固資訊
- **CPU**：Intel 標準保固
- **記憶體**：廠商保固
- **儲存**：廠商保固
- **系統**：整合保固

### 技術支援
- **硬體問題**：聯絡系統整合商
- **韌體更新**：定期檢查更新
- **相容性**：確認硬體相容性

## 資源分配規劃

### Kubernetes 集群資源
- **總 CPU 核心**：28 個邏輯處理器
- **總記憶體**：64 GB
- **儲存空間**：
  - 系統：512 GB SATA
  - 資料：2 TB NVMe

### VM 資源分配建議
- **Master 節點**：3 個 × (4/3 核心, 8GB RAM)
- **Worker 節點**：1 個 × (12 核心, 24GB RAM)
- **保留資源**：4 核心, 8GB RAM (系統管理)

## 效能基準

### CPU 效能測試
```bash
# CPU 壓力測試
stress-ng --cpu 28 --timeout 60
```

### 儲存效能測試
```bash
# NVMe 讀寫測試
fio --name=randwrite --rw=randwrite --bs=4k --size=1g --numjobs=4 --runtime=60 --directory=/mnt/nvme-test
```

### 網路效能測試
```bash
# 網路吞吐量測試
iperf3 -c target-ip -t 60
```
